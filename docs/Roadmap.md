# ARIA Roadmap

## Purpose

This roadmap outlines the planned evolution of ARIA.

It is intended to guide development, prioritize features, and prevent feature creep. Features may move between versions as the product evolves, but the focus of each release should remain consistent.

---

# Guiding Principle

Every release should improve one thing:

**Helping learners communicate naturally and confidently in another language.**

New features should only be added if they support this goal.

---

# Version 0.1 — The First Conversation

**Goal:**

A learner can visit an Irish pub, meet Maeve, and complete a short, enjoyable conversation using gradual immersion.

### Features

- ARIA framework
- Gaeilge learning experience
- The Hearth & Harp pub
- Maeve as conversational guide
- Typed learner responses
- AI-generated conversations
- Audio playback
- Replay button
- Slower playback
- Adaptive English support
- Basic learner progression
- Session memory

**Success Criteria**

A learner finishes their first conversation wanting to return tomorrow.

---

# Version 0.2 — Returning Visitor

**Goal:**

Create the feeling of returning to a familiar place.

### Planned Features

- Long-term learner memory
- Personalized greetings
- Conversation history
- Expanded pub dialogue
- Additional conversation topics
- Voice input
- Basic pronunciation feedback

---

# Version 0.5 — Living World

**Goal:**

Expand beyond the bar while preserving the conversational experience.

### Planned Features

- Additional pub characters
- Clickable objects
- Reading by the fireplace
- Listening activities
- Village events
- Expanded vocabulary contexts

---

# Version 1.0 — Complete Gaeilge Experience

**Goal:**

Provide a complete immersive Gaeilge learning environment.

### Planned Features

- Multiple conversational personas
- Rich learner memory
- Adaptive difficulty
- Full speech interaction
- Dynamic conversations
- Cultural experiences
- Achievements based on learning rather than streaks

---

# Future Languages

The ARIA framework is designed to support multiple languages while preserving the same learning philosophy.

Potential future experiences include:

- Japanese
- Spanish
- French
- Italian
- Scottish Gaelic
- Welsh
- Latin
- Biblical Hebrew
- Koine Greek

Each language should feature:

- Authentic environments
- Native conversational personas
- Culturally appropriate interactions
- Region-specific language support where practical

---

# Deferred Ideas

The following ideas are intentionally outside the initial releases.

- Multiplayer conversations
- AI-generated villages
- Character customization
- Multiple simultaneous locations
- Community-created content
- Marketplace features

These ideas may be revisited after the core learning experience has been validated.

---

# Product Philosophy

ARIA succeeds when learners return because they enjoy the conversations—not because they fear losing a streak.

Every development decision should reinforce this philosophy.
